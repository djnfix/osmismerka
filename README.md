# osmismerka

Interaktivní česká osmisměrka s diakritikou, dotykovým výběrem slov a 130 náhodně vybíranými těžkými levely 16 × 16.

Slova jsou ukrytá ve všech osmi směrech: vodorovně, svisle i diagonálně, vždy oběma směry. Zbylá písmena po přečtení po řádcích tvoří tajenku. Během hry se zobrazuje pouze její maska a celé řešení se odhalí po nalezení všech slov v zavíratelném dialogu.
